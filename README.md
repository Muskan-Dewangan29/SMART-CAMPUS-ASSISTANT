# 🎓 Smart Campus Assistant (SAC)

An AI-powered chatbot for SSIPMT that helps students with:

- 🎓 Admissions
- 📚 Courses
- 💰 Fees
- 🏠 Hostel
- 🎯 Placements
- 🎁 Scholarships
- 📞 Contact Information

## 🚀 Live Demo

https://sac-uyvi.onrender.com

## 🛠️ Tech Stack

- React + Vite
- Express.js
- Node.js
- Google Gemini API
- Render
- Tailwind CSS
